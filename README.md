# AppDev - Learn Application Development

An interactive learning platform for Application Development.

## Features

- 🔐 User authentication with Supabase
- 📚 5 learning rooms covering App Development topics
- 💻 Interactive code challenges
- 🖥️ Terminal simulator for command practice
- 🏆 XP and leveling system
- 📊 Leaderboard to compete with others
- 📱 Fully responsive design

## Setup Instructions

1. Create a Supabase account at https://supabase.com
2. Create a new project
3. Run the SQL setup script in the SQL Editor
4. Copy your Project URL and Anon Key
5. Update `js/supabase.js` with your keys
6. Deploy to GitHub Pages
